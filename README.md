## О проекте

**Corporate Assistant** — компактная основа корпоративного Telegram-бота, которую можно постепенно дополнять новыми рабочими сценариями. Сейчас проект помогает вести календарь отпусков, хранить профили сотрудников и вовремя доставлять персональные и общие уведомления.

Взаимодействие построено вокруг привычного интерфейса Telegram: команды, понятные сообщения и inline-кнопки. Для выбора отпуска используется календарная сетка с днями недели и отметками выходных.

## 🚀 Быстрый старт

### Требования

- Python **3.11–3.14**;
- PowerShell;
- токен Telegram-бота, полученный у [@BotFather](https://t.me/BotFather).

### 1. Подготовьте проект

Откройте PowerShell в корневом каталоге и выполните:

```powershell
powershell -ExecutionPolicy Bypass -File .\onBots.ps1
```

Скрипт автоматически:

1. создаст изолированное окружение `.venv`;
2. установит приложение и зависимости;
3. подготовит файл `.env`;
4. инициализирует SQLite-базу.

### 2. Добавьте токен

Откройте созданный файл `.env` и заполните значение:

```dotenv
TELEGRAM_BOT_TOKEN=ваш-токен
```

### 3. Запустите бота

```powershell
powershell -ExecutionPolicy Bypass -File .\onBots.ps1 -RunBot
```

После запуска откройте бота в Telegram, нажмите **Start** и следуйте подсказкам регистрации.

## 🛠️ Технологии

- **aiogram** — асинхронное взаимодействие с Telegram Bot API;
- **APScheduler** — фоновые напоминания и уведомления;
- **SQLite** — профили, отпуска и очередь сообщений;
- **Pydantic Settings** — конфигурация окружения;
- **Typer** — консольная точка входа;
- **Pytest + Ruff** — тестирование и контроль качества кода.

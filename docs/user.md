# Руководство пользователя

## Первое подключение

1. Откройте бота и отправьте `/start`.
2. Укажите имя в одном из форматов: `Фамилия Имя`, `Фамилия Имя Отчество` или `Фамилия И.О.`.
3. Укажите дату рождения в формате `ДД.ММ.ГГГГ`.

Бот сохраняет Telegram ID, тег `@username` и данные профиля. При повторном входе существующий пользователь, его роль, настройки и отпуска не удаляются.

## Команды

- `/vacation` — открыть календарь планирования отпуска;
- `/my_vacations` — показать свои отпуска;
- `/profile` — посмотреть персональные данные;
- `/help` — краткая справка.

В `/profile` под данными расположены кнопки с пометкой `✏️ Изменить`: ими редактируются ФИО, дата рождения, телефон и email.

В календаре `/vacation` столбцы идут с понедельника по воскресенье, выходные отмечены `🔴`. Пользователь может выбирать текущий и следующий год. Прошедшие даты добавляет администратор.

Бот уведомляет о начале отпуска за 14 дней и за 1 день. Общие корпоративные уведомления приходят автоматически в дату и время, заданные администратором.

# Руководство пользователя

## Первое подключение

1. Откройте бота и отправьте `/start`.
2. Укажите полное ФИО.
3. Укажите номер телефона и локацию пребывания.
4. Выберите грейд и направление.
5. Укажите рабочий Email.

Бот сохраняет Telegram ID и профиль. При повторном запуске данные, права, команда и отпуска сохраняются.

## Команды сотрудника

| Команда | Назначение |
|---|---|
| `/vacation` | Добавить отпуск через календарь |
| `/my_vacations` | Посмотреть и изменить свои отпуска |
| `/profile` | Посмотреть или отредактировать профиль |
| `/contacts` | Открыть контакты сотрудников |
| `/help` | Показать доступные команды |

Telegram показывает меню с учётом прав текущего пользователя. Служебные команды владельца сотрудникам не отображаются.

## Профиль

Обязательные данные: полное ФИО, телефон, локация пребывания, грейд, направление и рабочий Email.

Дополнительно можно указать личный Email (`Email_P`), уровень английского, дату трудоустройства, город офиса, формат работы, проект и дату старта на проекте.

## Отпуска

Сотрудник может создавать, изменять и удалять свои отпуска. Тимлид управляет календарями участников своей команды, владелец — календарями всех сотрудников.

Гость видит только свой календарь и контакт назначенного тимлида.

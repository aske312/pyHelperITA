# Руководство гостя

Гостевой доступ предназначен для знакомства с помощником и ограниченной работы без пароля сотрудника. После выбора гостевого входа бот показывает стартовое сообщение из `docs/guest_welcome.md`; его можно редактировать вручную без изменения Python-кода.

## Доступные команды

- `/vacation` — добавить собственный отпуск, если команда включена владельцем.
- `/profile` — открыть и отредактировать собственную карточку.
- `/help` — показать доступные команды и получить это руководство.

Гость не видит общий справочник контактов, события компании и административные инструменты. Для получения доступа сотрудника обратитесь к руководителю или владельцу продукта.

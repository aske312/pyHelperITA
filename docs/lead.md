# Руководство руководителя команды

## Команды

- /employees — карточка вашей команды, действия с сотрудниками и разовое оповещение команды.
- /invite_team — пригласить сотрудника.
- /dismiss_team — исключить сотрудника.
- /vacation и /my_vacations — управление собственными отпусками.
- /profile — личный профиль.
- /contacts — справочник сотрудников.
- /help — команды и это руководство.

Руководитель может управлять только своей командой. При переносе сотрудника из другой команды владелец продукта получает уведомление.

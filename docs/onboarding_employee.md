# Онбординг сотрудника

Добро пожаловать в корпоративного помощника. Сначала заполните профиль: ФИО, международный телефон, город, грейд, направление и рабочую почту. Значения со списком выбирайте кнопками, даты вводите как `ДД.ММ.ГГГГ`.

После заполнения откройте `/help`, чтобы получить руководство своей роли. Для первого отсутствия используйте `/absence`, выберите тип события и следуйте подсказкам. Команда `/clear` отменяет незавершённый сценарий и закрывает текущее меню.

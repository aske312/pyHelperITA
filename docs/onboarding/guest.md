# Онбординг гостя

Вы вошли с гостевым доступом. Заполните короткий профиль по подсказкам бота. Гость может оформить отпуск через `/absence`, открыть свои отпуска через `/my_events`, настроить профиль и интеграции.

Больничный, Day Off, общий справочник сотрудников и административные функции гостю недоступны. Команда `/help` показывает актуальное руководство, а `/clear` отменяет текущий ввод.

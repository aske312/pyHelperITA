# Документация Corporate Assistant

- `onboarding/` — первые шаги сотрудника и гостя;
- `guides/` — пользовательские руководства по ролям;
- `technical/` — архитектура, функции, интеграции, БД и конфигурация;
- `backlog/` — идеи и технический долг, не являющиеся действующей спецификацией.

Техническое чтение рекомендуется начинать с `technical/architecture.md`, затем
перейти к `technical/functionality.md` и `technical/database.md`. Развёртывание
описано в `technical/vps-deployment.md`.
